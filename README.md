## 📄 Industrial IoT with ML-Based Anomaly Detection for CNC Machines

This project presents an **Industrial IoT (IIoT) and Machine Learning-based solution** for detecting anomalies in CNC and metal manufacturing machines. The system focuses on improving **predictive maintenance, reducing downtime, and enhancing operational efficiency** in industrial environments.

---

### 🚀 Overview

Modern CNC machines generate large volumes of sensor data, but traditional monitoring systems fail to utilize this data effectively. This project integrates **IIoT and Machine Learning** to enable **real-time monitoring and intelligent fault detection**.

---

### ⚙️ Key Features

- 📡 Real-time CNC data acquisition using IIoT  
- 🧠 Machine Learning-based anomaly detection  
- 📊 Data preprocessing and feature engineering  
- ⚖️ Balanced dataset for improved model performance  
- 📈 Comparative analysis of multiple ML models  
- 🖥️ Dashboard for monitoring machine health and anomalies  

---

### 🧪 Machine Learning Models Used

- Feedforward Neural Network  
- Random Forest  
- Gradient Boosting  

All models were trained and evaluated on the same dataset to ensure fair comparison.

---

### 📊 Dataset Details

- ~10 lakh raw industrial records  
- Processed to **94,476 structured entries**  
- 19 optimized features  
- Time-series CNC sensor data  
- Balanced dataset (Normal vs Anomaly)  

---

### 🧩 Methodology

1. Data collection from CNC machine logs  
2. Data cleaning and preprocessing  
3. Feature engineering (time features + rolling averages)  
4. Model training and evaluation  
5. Performance comparison  
6. Deployment-ready monitoring system  

---

### 📈 Results & Insights

- Gradient Boosting showed the **best overall performance**  
- Tree-based models performed better on structured industrial data  
- Key influencing features: **Speed, Load, Position parameters**  
- System effectively detects faults with minimal false alarms
